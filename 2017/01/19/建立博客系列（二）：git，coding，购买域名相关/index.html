建立博客系列（二）：git，coding，购买域名相关

发表于 2017-01-19 | 更新于 2022-11-17 | 分类于 建立博客 | 阅读次数：

此系列教程采用hexo+coding搭建博客，所以首先介绍下git和coding。

git

搭建博客使用的git知识不多，这里仅介绍我们需要的知识，git的详细知识推荐
廖雪峰老师的git教程
git多用于版本管理，多人合作，但如果用hexo写博客，git的命令几乎不用。

安装git,这里仅介绍windows的

首先在 git下载 ， 选择最新版的 64-bit Git for Windows Setup ，如果电脑是32位的找到下面的 32-bit Git for Windows Setup，下载即可。
安装过程中，安装位置自己选择，还有就是把添加到桌面勾选上

其他一路Next就可以了。

git默认路径调整

打开git bash,输入命令pwd(Print working directory),显示当前路径，我们可以看到是在C盘里。

虽然现在硬盘空间很大了，放到C盘也不影响什么，但一些软件增加删除会引起文件碎片化严重，影响C盘文件的读写速度，还有就是为了数据的安全，以免忘记备份，我们现在将它的默认路径改到非系统盘。我以D盘为例讲一下。

3.运行命令

4.在D盘新建文件夹，取名git
5.找到pwd后显示的路径，我的是c:\用户\asus,然后将其目录下的.gitconfig复制到上一步建立的文件夹中

6.配置环境变量，以win7 为例，在我的电脑点击右键，属性，高级系统设置，环境变量，在系统变量里看一下有没有HOME变量，没有的话自己新建一个，并且添加值 D:\git

Coding

Coding可以类比于国外的github，可以理解为一个远程仓库，可以把我们的代码什么的放进去，这里之所以用到它，我们可以把文章放进去，这样的话，就不需要再额外购买服务器了，由于是国内的，访问速度也比github强。

注册

进入 Coding ，按照提示注册即可。

创建一个项目

点击添加项目

输入名称blog,其他不用改，点击创建项目即可。

添加SSH公钥

我们从Coding克隆一个项目可以用https方式和SSH两种，https每次操作都需要输入Coding的账号和密码，比较繁琐，所以我们采用SSH方式。

1.打开gitbash,输入命令 ssh-keygen -t rsa -C “your_email@youremail.com“
双引号里的邮箱改成你自己的邮箱，然后一路回车，会出现一张萌萌哒图

2.打开D:\git\ssh目录，用文本编辑器打开id_rsa.pub,将里边的内容复制

3.打开Coding,点击账户，SSH公钥，将刚才复制的内容加到公钥内容中，名称自己写。

4.打开gitbash,输入命令 ssh -T git@git.coding.net，中间需要输入yes,然后出现successfully即代表配置成功。

购买域名

不购买域名的话，以后访问自己的博客会是这样的链接 wind_liang.coding.me/blog， 好长好长，购买了域名就可以关联成自己购买的例如我的 windliang.cc
我知道购买域名的地方有两家，腾讯云，阿里云，我开始买的是腾讯云，但免费的DNS解析比较慢，花钱的DNS又比较贵，而阿里花钱的DNS解析还可以接受，所以域名解析我用的阿里的，所以后边教程也将以阿里云的域名进行讲解。
如果大家准备购买的话，点开链接后注册账号即可，然后输入自己想购买的域名，再选择自己能接受的价格购买即可，购买的时候会提示实名认证，按流程进行即可，实名认证通过一般为3到5天。
